Frontend pertemuan_07 — sederhana berdasarkan slide

Quick start:

1. Set environment variable `MONGODB_URI` to your MongoDB connection.
2. (Optional) set `BASE_URL` to `http://localhost:3000` if not default.

Install and run:

```bash
npm install
npm run dev
```

API:
- `GET /api/products` — daftar produk
- `GET /api/products/:id` — detail produk

Notes:
- This is a minimal scaffold taken from lecture slides. Fill sample products into MongoDB `ecommerce.products` collection to see data.
